服务器ip和端口号为: 104.224.129.246 : 10000

目前有两种响应方式：
1. 连接成功后，用户发送单字节信息"d"，服务器接收后将发送一个四字节的信息表示准备发送给客户端的文件数目number，接下来是number个文件的信息，首先是一个128字节的字节串表示文件名，然后是4字节表示文件大小k，接下来k个字节为文件内容。重复number次发送信息后服务器将终止该次响应。
2.连接成功后，用户发送单字节信息"u"，服务器接收后将准备接受用户发来的信息，用户需要发送一个128字节的字节串表示文件名，然后是4字节表示文件大小k，用户接下来需将文件的k个字节发送给服务器。

注意，这里设置了发送文件时字节串的大小为1024个byte，理论上是不需要这么麻烦的，但是据说可以减轻服务器压力（服务器压力也不大），也不容易丢包（别问我为什么，我计算机网络没学好，瞎说的）。

csharp文件夹下的文件client.cs是和python中的client_copy.py作用相似的,供参考。
